## Installation / Installation / 安装
1. Repository klonen:
```bash
git clone https://github.com/Tobias-Bao/Tobias-Bao.github.io.git
```
2. Projekt öffnen:
```bash
cd Tobias-Bao.github.io
```
3. Im Browser öffnen:
```bash
start index.html
```
