# 🚀项目描述

## 📁概述

- `css/`: 样式文件
  - `currency-converter.css`: 汇率转换器样式
  - `global.css`: 全局样式
- `images/`: 图片资源
- `js/`: 动画文件
  - `currency-converter.js`: 汇率转换器动画
  - `global.js`: 全局动画
- `partials/`: 功能页
  - `currency-converter.html`: 汇率转换器页
- `index.html`: 主页

## ✨功能

- 💱 汇率转换
- 📱 响应式设计
